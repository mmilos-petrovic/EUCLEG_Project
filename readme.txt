Source code for statistical analysis of data from the EUCLEG project.

The experiment is based on augmented design, which included 9 environments in 4 countries and 400 faba bean genotypes.

The following statistical analyses were used:
  1. Cluster analysis
  2. Mixed model design
  3. Which-won-where analysis
  4. Mean vs. stability analysis
  5. Mixed model analysis
  6. Principal component analysis
